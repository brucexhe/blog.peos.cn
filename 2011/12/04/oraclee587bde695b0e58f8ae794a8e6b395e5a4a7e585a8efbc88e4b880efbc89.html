 68. TRANSLATE(string,from_str,to_str)

返回 string 经过处理后的字符串。处理规则为：在 from_str中出现的字符，都替换为对应位置的 to_str中的字符。未在 from_str中出现的字符不被替换。

参数 from_str中可以包含比 to_str多的字符，此时，from_str中额外的字符在 to_str中没有与之对应的字符，若这些字符在 string中出现，它们将会从 string中删除。

在参数 to_str中，你不能使用空字符串来试图删除在 from_str中存在的所有字符。Oracle 会把空字符串作为 NULL 处理，此时会返回 NULL。

SELECT   TRANSLATE('abcdefghij','abcdef','123456') result   FROM   dual;
result
--------------
123456ghij

SELECT   TRANSLATE('abcdefghij','abcdefghij','123456') result   FROM   dual;
result
----------
123456
